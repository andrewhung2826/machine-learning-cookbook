# :book: Machine Learning CookBook

A practical "cookbook" featuring clean, easy-to-follow implementations of Machine Learning and Deep Learning algorithms.

This repository is built with a dual-focus approach:
1. **From Scratch (NumPy):** Implementing core algorithms from the ground up to deeply understand the underlying mathematics, matrix operations, and optimization techniques (e.g., Gradient Descent).
2. **Library-based:** Utilizing powerful modern libraries (Scikit-Learn, PyTorch, Pandas) to train the model.

## :file_folder: Repository Structure
```text
machine-learning-cookbook/
│
├── algorithms/              # Machine Learning algorithm implementations
│   ├── from_scratch/        # Implementing core algorithms from zero using NumPy
│   └── from_library/        # Implementing algorithms using modern frameworks (Scikit-Learn, Pytorch, etc.)
│
├── data/                    # Raw and dummy datasets used for testing models
│
└── library/                 # Comprehensive tutorials for core data science libraries
    ├── numpy/
    ├── pandas/
    └── pytorch/
```

## :rocket: Getting Started
Step 1: Clone the repository
```bash
git clone [https://github.com/andrewhung2826/machine-learning-cookbook.git](https://github.com/andrewhung2826/machine-learning-cookbook.git)
cd machine-learning-cookbook
```
Step 2: Set up a virtual environment (`venv`) and install dependencies
```bash
# Create a virtual environment (Python 3.12 recommended)
conda create -p venv python==3.12 -y

# Activate the venv 
conda activate venv/

# Upgrade pip and install the required packages
pip install --upgrade pip
pip install -r requirements.txt
```
